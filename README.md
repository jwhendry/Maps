# Real Code example
```java
		List<String> list = Arrays.asList("one", "two", "three");
		List<Integer> integers = Arrays.asList(1,2,3);
		
		Map<String, Integer> map = new HashMap<>();
		
		for(int i = 0; i< list.size();++i) {
			map.put(list.get(i), integers.get(i));
		}
		System.out.println(map);
	}
```
